## 📌 Overview
**Solar Energy Yield Forecasting** is a comprehensive full-stack web application designed to predict and analyze solar power generation based on historical weather and energy data. Built with the **MERN stack** and integrated with a **Python-based prediction model**, this platform provides interactive data visualizations and accurate yield forecasting to help optimize solar energy utilization.

## ✨ Key Features
* **Accurate Forecasting:** Utilizes Python to process environmental data and predict solar energy yield.
* **Interactive Dashboard:** A highly responsive and user-friendly UI built with React.js for data visualization.
* **Robust API Architecture:** Engineered a secure backend using Node.js and Express.js to construct RESTful APIs.
* **Efficient Data Management:** Designed a scalable MongoDB database schema to securely store and retrieve large volumes of historical data.

## 🛠️ Tech Stack
* **Frontend:** React.js, HTML5, CSS3, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Prediction Model:** Python
* **Deployment:** PythonAnywhere 

## 🏗️ System Architecture
1. **Client Interface:** React.js dashboard captures user requests and displays forecast metrics.
2. **API Layer:** Express.js securely routes requests to the appropriate backend services.
3. **Data Processing:** Python scripts process historical datasets and generate predictions.
4. **Storage:** MongoDB handles the efficient storage and retrieval of environmental variables.
